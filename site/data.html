Application Data

The countries and US-states population information were obtained from the United States Census Bureau website. Specifically, from the following links:

  • Annual Estimates of the Civilian Populations by Single Year of Age and Sex for the United States and States: April 1, 2010 to July 1, 2014
  • International Database: Population per single year report from 2010 to 2014 for Peru, Uruguay, Japan, Germany and Mexico

    Differences and Project Decisions
    United States & States Dataset International Database Project Decision
    Year data using a columns approach Year data using a rows approach Year data using a columns approach
    Sex data using a rows approach Sex data using a colums approach Sex data using a rows approach. However, only took into account registers for totals males and females.
    Yearly age distribution for people 85 years old and more is summed in a 85+ register. Used the following groups for elder people: 85-90, 90-00, 100+. Group these data in a range 85+ as there wasn't any way to get yearly information from both datasets. As a consequence, the bar chart always shows a bigger bar for 85+ when using the "1 Year Range".
    Total population for a given place/year used the Age = 999 filter Total population for a given place/year used the Age = Total filter Age = 999 filter
  2. The website allows the user to download data with 1 Year Range or 5 Years Range. However, the requirement was to show the data in a 10 Years Range.
    • To format the United States & States dataset, I developed a program in python that received the file in its original format and created a new worksheet with the data formatted.
    • For python I used the openpyxl library
    • To format the International Database dataset, I used MS Excel tools.
